# Nola

**N**atural **O**n-device **L**anguage **A**ssistant — a macOS app for chatting with LLMs running locally on your Mac.

Built with SwiftUI and [MLX](https://github.com/ml-explore/mlx-swift) for Apple Silicon. Models run entirely on-device.

## Requirements

- macOS 26 (Tahoe)
- Apple Silicon Mac
- Xcode 26

## Build

```
open Nola.xcodeproj
# Cmd+R
```
